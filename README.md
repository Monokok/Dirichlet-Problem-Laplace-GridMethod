# Dirichlet Problem for Laplace Equation (FDM)

Описание задачи
В проекте реализовано приближённое решение **задачи Дирихле для уравнения Лапласа**  
в квадратной области методом сеток с использованием С++, MPI.

Рассматривается квадрат **ABCD** с вершинами:
- A(0, 0)
- B(0, 100)
- C(100, 100)
- D(100, 0)

Численное решение строится с использованием **формул Либмана**
(итерационный метод).

---

Параметры расчёта
- Метод: **метод сеток (Finite Difference Method)**
- Итерационный метод: **метод Либмана**
- Шаг сетки: `h = 0.001`
- Максимальное число итераций: `Imax = 1000`

---

В рамках работы выполнено:
- Решение задачи Дирихле для уравнения Лапласа методом Либмана
- Реализация **последовательного алгоритма**
- Реализация **параллельного алгоритма с использованием MPI**
- Использование **конвейерного вида параллелизма**
